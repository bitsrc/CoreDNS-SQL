# CoreDNS-sql

A Docker container project that combines CoreDNS with the pdsql plugin and PostgreSQL bindings

## Overview

This project packages CoreDNS alongside pdsql, enabling DNS queries to be resolved through a PostgreSQL database backend.

## Features

- **CoreDNS**: Modern DNS server written in Go
- **pdsql**: PostgreSQL backend plugin for CoreDNS
- **PostgreSQL**: Database driver
